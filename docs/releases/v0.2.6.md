# Codex LCM v0.2.6

`v0.2.6` adds cross-project session observability and avoids unnecessary raw-log
scans during normal hook ingestion.

## Changes

- ([#42](https://github.com/Team-Volt/codex-lcm/pull/42)) Add session listing and token usage to the CLI and MCP server, including parent/child lineage, model and reasoning metadata, cumulative token usage, and safe overflow capture for oversized hook inputs.
- ([#43](https://github.com/Team-Volt/codex-lcm/pull/43)) Record synchronized raw-log state in SQLite so unchanged logs skip full rescans while edited or truncated logs still use the reconciliation path.

## Upgrade to v0.2.6

Restart Codex Desktop or open a fresh Codex CLI/TUI session after upgrading so
the refreshed plugin cache, MCP server, hooks, and skill are loaded.

For an existing GitHub marketplace install:

```sh
codex plugin marketplace remove codex-lcm
codex plugin marketplace add Team-Volt/codex-lcm --ref v0.2.6
codex plugin add codex-lcm@codex-lcm
```

For a local checkout install:

```sh
git -C /path/to/codex-lcm fetch --tags origin
git -C /path/to/codex-lcm checkout v0.2.6
codex plugin add codex-lcm@codex-lcm
```
