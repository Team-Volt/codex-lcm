# Codex LCM v0.2.4

`v0.2.4` improves session capture and recall reliability, including work
performed by spawned agents.

## Fixes

- ([#35](https://github.com/Team-Volt/codex-lcm/pull/35)) Preserve indexed evidence during malformed-log recovery, prevent duplicate concurrent writes, import current Codex session records, validate MCP requests, and prefer stronger exact recall matches.
- ([#36](https://github.com/Team-Volt/codex-lcm/pull/36)) Capture completed spawned-agent transcripts through the `SubagentStop` hook.

## Upgrade to v0.2.4

Restart Codex Desktop or open a fresh Codex CLI/TUI session after upgrading so
the refreshed plugin cache, MCP server, hooks, and skill are loaded.

For an existing GitHub marketplace install:

```sh
codex plugin marketplace remove codex-lcm
codex plugin marketplace add Team-Volt/codex-lcm --ref v0.2.4
codex plugin add codex-lcm@codex-lcm
```

For a local checkout install:

```sh
git -C /path/to/codex-lcm fetch --tags origin
git -C /path/to/codex-lcm checkout v0.2.4
codex plugin add codex-lcm@codex-lcm
```
