# Codex LCM v0.2.2

`v0.2.2` is a patch release for compaction recovery and recall safety.

## Fixes

- ([#30](https://github.com/Team-Volt/codex-lcm/pull/30)) Recover LCM context after compaction and continue unfinished work.
- ([#29](https://github.com/Team-Volt/codex-lcm/pull/29)) Nudge agents to use LCM recall after compaction.
- ([#28](https://github.com/Team-Volt/codex-lcm/pull/28)) Preserve token metrics during redaction.
- ([#27](https://github.com/Team-Volt/codex-lcm/pull/27)) Harden LCM recall output.

## Upgrade to v0.2.2

Restart Codex Desktop or open a fresh Codex CLI/TUI session after upgrading so
the refreshed plugin cache, MCP server, hooks, and skill are loaded.

For an existing GitHub marketplace install:

```sh
codex plugin marketplace remove codex-lcm
codex plugin marketplace add Team-Volt/codex-lcm --ref v0.2.2
codex plugin add codex-lcm@codex-lcm
```

For a local checkout install:

```sh
git -C /path/to/codex-lcm fetch --tags origin
git -C /path/to/codex-lcm checkout v0.2.2
codex plugin add codex-lcm@codex-lcm
```
