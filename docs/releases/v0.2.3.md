# Codex LCM v0.2.3

`v0.2.3` is a patch release for active-thread context packing.

## Fixes

- ([#32](https://github.com/Team-Volt/codex-lcm/pull/32)) Resolve active thread context packing when `CODEX_THREAD_ID` differs from the stored session ID.

## Upgrade to v0.2.3

Restart Codex Desktop or open a fresh Codex CLI/TUI session after upgrading so
the refreshed plugin cache, MCP server, hooks, and skill are loaded.

For an existing GitHub marketplace install:

```sh
codex plugin marketplace remove codex-lcm
codex plugin marketplace add Team-Volt/codex-lcm --ref v0.2.3
codex plugin add codex-lcm@codex-lcm
```

For a local checkout install:

```sh
git -C /path/to/codex-lcm fetch --tags origin
git -C /path/to/codex-lcm checkout v0.2.3
codex plugin add codex-lcm@codex-lcm
```
