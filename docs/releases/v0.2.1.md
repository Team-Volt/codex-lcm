# Codex LCM v0.2.1

`v0.2.1` is a patch release focused on hardening, performance, and release
cleanup after the first tagged developer release.

## Features

- ([#23](https://github.com/Team-Volt/codex-lcm/pull/23)) Add LCM context planning and file-reference support.

## Fixes

- ([#24](https://github.com/Team-Volt/codex-lcm/pull/24)) Harden storage redaction, repair raw-log index drift, and support framed MCP stdio.

## Performance

- ([#22](https://github.com/Team-Volt/codex-lcm/pull/22)) Speed up Codex session imports and write indexing.

## Maintenance

- ([#25](https://github.com/Team-Volt/codex-lcm/pull/25)) Remove stale cleanup surfaces.
- ([#21](https://github.com/Team-Volt/codex-lcm/pull/21)) Simplify the CLI JSON output helper.
- ([#20](https://github.com/Team-Volt/codex-lcm/pull/20)) Fix the `v0.2.0` release notes.

## Upgrade to v0.2.1

Restart Codex Desktop or open a fresh Codex CLI/TUI session after upgrading so
the refreshed plugin cache, MCP server, hooks, and skill are loaded.

For an existing GitHub marketplace install:

```sh
codex plugin marketplace remove codex-lcm
codex plugin marketplace add Team-Volt/codex-lcm --ref v0.2.1
codex plugin add codex-lcm@codex-lcm
```

For a local checkout install:

```sh
git -C /path/to/codex-lcm fetch --tags origin
git -C /path/to/codex-lcm checkout v0.2.1
codex plugin add codex-lcm@codex-lcm
```
